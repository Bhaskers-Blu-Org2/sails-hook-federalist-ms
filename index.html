sails-hook-federalist-ms

This installable hook for Sails.js incorporates both Windows and Microsoft Azure extensibility in to 18F's Federalist platform. The hook is comprised of two components: build support for Windows and static site publishing support for Azure.

Developing

This hook is developed following the Sails.js hook specification. Execute the following command to install the package's dependencies:

npm install

Unit tests are developed using Mocha which can be installed by running npm install -g mocha.

Gulp is used to execute all unit tests and compile the documentation from the code. Execute the following command to initiate the build tasks:

gulp
